Prompt Maestro para el Desarrollo de la App: "Erika del Castillo - Art Studio"
Rol: Actúa como Diseñador de Producto Senior y Desarrollador Full-Stack especializado en e-commerce de lujo y soluciones de Inteligencia Artificial Generativa.

Objetivo: Crear una aplicación móvil (iOS/Android) y Web que sirva como galería, tienda y herramienta de codiseño para el emprendimiento de esculturas de Erika del Castillo.

1. Identidad Visual y Experiencia de Usuario (UX/UI)
Estética: Estilo "Gallery-Zen". Fondo blanco puro o gris muy tenue, tipografías elegantes (Serif para títulos, Sans-serif ligera para cuerpos).

Inspiración: Basarse en el perfil de Instagram @esculturas_erikadelcastillo. Las fotos deben ser las protagonistas.

Interacciones: Transiciones fluidas, carga progresiva de imágenes y un visor de alta calidad que permita hacer zoom para apreciar la textura de las piezas.

2. Módulos de la Aplicación
A. Galería de Exhibición (Portfolio):

Feed infinito con las obras. Cada obra incluye: Nombre, materiales (ej. cerámica, acabados en oro, madera), dimensiones y una breve descripción emocional.

Categorías: Animales, Corazones, Abstractos, Colecciones Especiales.

B. Tienda Online (Stock Inmediato):

Pasarela de pagos integrada (Stripe/PayPal/Mercado Pago).

Cálculo automático de envíos.

Sistema de "Pieza Única": Cuando se vende, aparece automáticamente como "Vendida" en la galería para mantener el valor de exclusividad.

C. Sección de Pedidos Personalizados (Custom Commissions):

Formulario interactivo para que el cliente solicite una obra a medida.

Campos: Tamaño deseado, paleta de colores, materiales y fecha de entrega.

D. Estudio de Diseño IA (Foto a Escultura):

Funcionalidad: El usuario sube una foto (ej. su mascota o un retrato).

Lógica de IA: Utilizar una integración con la API de Stable Diffusion (ControlNet) o Midjourney.

Instrucción de estilo (System Prompt de IA): "Transforma la imagen cargada por el usuario en una escultura fiel al estilo de Erika del Castillo: formas redondeadas, minimalistas, superficies lisas y pulidas, con pequeños acentos de color o elementos dorados minimalistas".

Resultado: Generar una previsualización 2D de cómo se vería la escultura para que el cliente la use como referencia al hacer su pedido.

3. Stack Tecnológico Sugerido
Frontend: React Native (para app móvil) o Next.js (para web).

Backend: Firebase o Supabase (para gestión de usuarios, base de datos y almacenamiento de fotos).

IA: API de Replicate para ejecutar modelos de imagen a imagen.

CMS: Sanity.io o Strapi para que el artista suba fotos nuevas sin saber programar.

4. Requerimiento de Entrega
"Proporcióname la estructura de la base de datos, el flujo de navegación (User Flow) y el código inicial para la integración de la cámara en la sección de diseño IA."

Consejos adicionales para tu proyecto:
Aprovecha el estilo: Como las piezas de Erika suelen tener detalles pequeños (un pajarito, un corazón), asegúrate de que la sección de "Diseño con foto" tenga un botón de "Añadir detalle distintivo" donde el usuario pueda elegir ponerle el icónico corazón rojo a su diseño.

Realidad Aumentada (Opcional): Dado que son objetos decorativos, añadir una función de AR (Realidad Aumentada) para "Ver en mi mesa" sería un diferencial enorme para las ventas.

Marketing: En la sección de ventas a pedido, recalca que son piezas hechas a mano, ya que el valor del emprendimiento de Erika es lo artesanal y artístico frente a lo industrial.
